[B]Version 1.02[/B] | API 101032 | by olegbl[/CENTER]

[CENTER]A helper library for Elder Scrolls Online that provides structured information for points of interest, including category, completion state, and descriptive details.[/CENTER]

[SIZE="4"][COLOR="#3399FF"][B]What It Does[/B][/COLOR][/SIZE]

[LIST]
[*]Converts a zone POI into a category object with icon data.
[*]Returns whether a POI is complete.
[*]Returns descriptive text for a POI.
[*]Includes extra description support for special POIs such as Mundus Stones and Crafting Stations.
[/LIST]

[SIZE="4"][COLOR="#3399FF"][B]API[/B][/COLOR][/SIZE]

[CODE]
local poiCategories = LibPOI:GetPOICategories()
local poiCategory = LibPOI:GetPOICategory(zoneIndex, poiIndex)
local isComplete = LibPOI:IsComplete(zoneIndex, poiIndex)
local description = LibPOI:GetDescription(zoneIndex, poiIndex)

poiCategory.id
poiCategory.categoryName
poiCategory.completeIcons[1]
poiCategory.incompleteIcons[1]
[/CODE]

[SIZE="4"][COLOR="#3399FF"][B]Use Cases[/B][/COLOR][/SIZE]

[*]Zone exploration helpers
[/LIST]
