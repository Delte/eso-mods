[B]GamePadHelper[/B]
Version 1.06.10 | API 101049

A modular collection of quality-of-life improvements for Elder Scrolls Online, built for gamepad and console UI.

Open settings from:
[B]Options > GPH Settings[/B]

[B]Required Libraries[/B]
[LIST]
[*]LibCovetousCountess
[*]LibItemLinkDecoder
[*]LibTraitResearch
[*]LibMultiIcon
[/LIST]

[B]Highlights[/B]
[LIST]
[*]Gamepad and console UI focused.
[*]Each feature can be enabled or disabled individually.
[*]Supports all 7 official ESO client languages: EN, DE, FR, ES, RU, JP, ZH.
[*]Supports market price data when compatible price addons or data packages are installed.
[/LIST]

[B]Main Features[/B]
[LIST]
[*]Tooltip Price: cleaner item price display with supported market data.
[*]Tooltip Enchantment: cleaner enchantment text.
[*]Tooltip Poison: improved poison tooltip formatting.
[*]Tooltip Trait: better trait visibility in tooltips.
[*]Tooltip Font: cleaner item tooltip font for gamepad play.
[*]Inventory Traits: research trait indicators in inventory.
[*]Inventory Covetous Countess: highlights relevant Covetous Countess treasures.
[*]Gear Comparison: side-by-side item stat comparison panels.
[*]Provisioning Filter: hides low-level recipes below CP160.
[*]Fishing: controller vibration, reel alert, automatic bait selection, and fallback bait.
[*]Teleporter: BeamMeUp integration for map and chat jump helpers.
[*]Map Search: search wayshrines, zones, houses, and points of interest from the gamepad world map.
[*]Dungeon Finder: shows pledge quest names in the Dungeon Finder list.
[*]Overview Panel: root menu overview with quest details, daily reminders, daily writ tracking, companion rapport dailies, research, surveys, treasure maps, and antiquity leads.
[*]Loot Offset: moves the loot history panel upward so it does not overlap chat. Requires UI reload after toggling.
[*]Auto Charge: recharges equipped weapons after combat. Default threshold: 25%.
[*]Auto Repair: repairs equipped items at merchants when repair is available.
[*]Auto Eye: automatically slots and activates Antiquarian's Eye when usable.
[/LIST]

[B]Trait Icon Color Meaning[/B]
[LIST]
[*][COLOR="#00FF00"]Green[/COLOR]: Only accessible copy with this trait; safe to research.
[*][COLOR="#FFFF00"]Yellow[/COLOR]: Another copy with the same trait exists in your inventory.
[*][COLOR="#FF4444"]Red[/COLOR]: Another copy with the same trait exists in your bank.
[*][COLOR="#FFFFFF"]White[/COLOR]: Equipped item uses ESO's default magnifying glass icon; trait details are shown in the tooltip.
[/LIST]

[B]Optional Integrations[/B]
[LIST]
[*]TamrielTradeCentre: optional PC market price source.
[*]BeamMeUp: enables Teleporter functionality.
[*]LibPriceCache: optional price cache source.
[*]TSCPriceDataAPIXBNA, TSCPriceDataAPIPSNA, TSCPriceDataAPIXBEU, TSCPriceDataAPIPSEU: optional console price data packages.
[/LIST]

[B]Price Data Attribution[/B]
PC market data can use TamrielTradeCentre when installed:
https://www.esoui.com/downloads/info1245-TamrielTradeCentre.html

Console price data provided by Tamriel Savings Co Price Fetcher / TSC Price Data API packages when installed:
https://tamrielsavings.com/price-fetcher

[B]Notes[/B]
[LIST]
[*]Optional integrations improve specific features but are not required for GamePadHelper to load.
[*]If no supported market addon or price data package is installed, Tooltip Price still reformats basic price information.
[*]This addon is provided as-is. Please report issues on the GamePadHelper project page.
[/LIST]
