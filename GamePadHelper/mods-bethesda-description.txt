# GamePadHelper

A modular collection of quality-of-life improvements for Elder Scrolls Online, built for gamepad and console UI. Every feature can be toggled individually from Options > GPH Settings.

## Features

- Overview Panel — two-column root menu with quest details and daily reminders.
- Map Search — search wayshrines, zones, houses, city services, daily quest givers, travel NPCs, and crafting stations. In Cyrodiil, shows accessible keeps with group proximity info. While dead, switches to revive mode and lists all revivable locations including forward camps.
- Teleporter — world map hotkey to teleport to a hovered zone.
- Dungeon Finder — shows pledge quest names in the finder list.
- Fishing — vibration on bite, reel alert, automatic bait selection and fallback.
- Auto Repair — repairs equipped gear at merchants automatically.
- Auto Charge — recharges weapons in and out of combat using regular soul gems.
- Antiquarian's Eye — auto-slots and activates the collectible when appropriate.
- Provisioning Filter — hides low-level recipes below CP160.
- Gear Comparison — side-by-side item stat comparison panels.
- Loot Offset — moves the loot history panel up to avoid chat overlap.
- Tooltip improvements — price, enchantment, poison, trait, and font.
- Inventory Trait — research indicators (blue/green/yellow/red) with duplicate counts and deconstruction legend.
- Inventory Countess and Bursar — highlights treasures for The Covetous Countess and Bursar of Tributes quests.

## Required Libraries

- LibCovetousCountess
- LibItemLinkDecoder
- LibTraitResearch

IMPORTANT: If you installed LibMultiIcon for older GamePadHelper builds, remove or disable it. GamePadHelper now includes its own private multi-icon helper.

## Optional Integrations

Market prices via TamrielTradeCentre (PC) or Tamriel Savings Co Price Fetcher / TSC Price Data API packages (console).
