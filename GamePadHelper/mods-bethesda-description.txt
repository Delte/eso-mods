# GamePadHelper

A modular collection of quality-of-life improvements for Elder Scrolls Online, built for gamepad and console UI.

Open settings from:
Options > GPH Settings

## What's New in 1.06.14

- API 101050 update: updated for the latest ESO patch.
- Per-character settings: GamePadHelper settings and Map Search save data now store per character, while the shared bookmark pool remains available when enabled.
- Trait indicator refresh: equipped researchable items now use a blue state, non-equipped items keep green/yellow/red duplicate logic, and tooltip trait explanations were refreshed to match.
- Deconstruction trait legend: deconstruction screens now show a right-side icon legend for research, duplicate counts, ornate, and intricate markers.
- Countess and Bursar split: The Covetous Countess and Bursar of Tributes now have separate inventory and tooltip toggles.
- Shared quest icon styling: Countess and Bursar now use the same quest icon, with green for active Countess items and blue for active Bursar items.
- Updated treasure checks: Countess and Bursar treasure detection now follows the updated LibCovetousCountess logic and active quest detection.

## Required Libraries

- LibCovetousCountess
- LibItemLinkDecoder
- LibTraitResearch

IMPORTANT: If you installed LibMultiIcon for older GamePadHelper builds, remove or disable it. GamePadHelper now includes its own private multi-icon helper.

## Trait Icon Color Meaning

- Blue: equipped item with a researchable trait.
- Green: only accessible copy with this trait.
- Yellow: have duplicate in inventory.
- Red: have duplicate in bank.
- Deconstruction screens include a right-side legend panel for trait colors, duplicate counts, and ornate/intricate icons.
