[CENTER][SIZE="5"][COLOR="#3399FF"][B]GamePadHelper[/B][/COLOR][/SIZE]
[B]Version 1.06.13[/B] | API 101049 | by olegbl, quelron[/CENTER]

[CENTER]A modular collection of quality-of-life improvements for Elder Scrolls Online, built specifically for gamepad and console UI.[/CENTER]

[SIZE="4"][COLOR="#3399FF"][B]What's New In 1.06.13[/B][/COLOR][/SIZE]

[LIST]
[*][B]Notifications entry[/B] - What's New now appears in the gamepad Notifications menu instead of using an automatic popup.
[*][B]Inventory Trait refresh[/B] - bag, bank, and deconstruction trait icons now update colors, counts, and icon switching more reliably.
[*][B]Private Multi-Icon helper[/B] - GamePadHelper no longer depends on [B]LibMultiIcon[/B]. If you installed the standalone library for older builds, remove or disable it.
[*][B]Map Search cache reuse[/B] - city service scan data is now reused across ReloadUI and relogin, and refreshes only after ESO API updates or when the cache is cleared.
[*][B]Auto Charge rework[/B] - weapons now recharge in and out of combat using regular soul gems.
[/LIST]

[SIZE="4"][COLOR="#3399FF"][B]Features[/B][/COLOR][/SIZE]

[LIST]
[*][B]Overview Panel[/B] - quest details on the left and daily reminders on the right.
[*][B]Map Search[/B] - search wayshrines, zones, houses, city services, daily quest givers, and travel NPCs.
[*][B]Teleporter[/B] - world map teleport hotkey and chat jump options.
[*][B]Dungeon Finder[/B] - shows pledge quest names in the finder list.
[*][B]Fishing[/B] - vibration, reel alert, automatic bait selection, and fallback bait support.
[*][B]Auto Repair[/B] - repairs equipped gear at merchants.
[*][B]Auto Weapon Charge[/B] - recharges weapons in and out of combat.
[*][B]Antiquarian's Eye[/B] - auto-slots and uses the collectible when appropriate.
[*][B]Provisioning Filter[/B] - hides low-level recipes below CP160.
[*][B]Gear Comparison[/B] - side-by-side item comparison panels.
[*][B]Loot Offset[/B] - moves the loot history panel upward to avoid chat overlap.
[*][B]Tooltip Price / Enchantment / Poison / Trait / Font[/B] - cleaner tooltip presentation for gamepad.
[/LIST]

[SIZE="4"][COLOR="#3399FF"][B]Inventory Trait Colors[/B][/COLOR][/SIZE]

[LIST]
[*][COLOR="#00FF00"]Green[/COLOR] - only accessible copy with this trait; safe to research.
[*][COLOR="#FFFF00"]Yellow[/COLOR] - another copy with the same trait exists in your inventory.
[*][COLOR="#FF0000"]Red[/COLOR] - another copy with the same trait exists in your bank.
[/LIST]

[SIZE="4"][COLOR="#3399FF"][B]Important[/B][/COLOR][/SIZE]

[LIST]
[*]If you installed [B]LibMultiIcon[/B] for older GamePadHelper builds, remove or disable it. GamePadHelper now uses its own private multi-icon helper.
[/LIST]
